# Browser Monitor 1.0.0

The first public release of Browser Monitor: local tab activity insights and browser-level protection in one Chrome extension.

## Install

1. Download `browser-monitor-1.0.0.zip` below.
2. Extract the archive to a permanent folder.
3. Open `chrome://extensions` and enable **Developer mode**.
4. Select **Load unpacked** and choose the extracted folder containing `manifest.json`.
5. Pin Browser Monitor to the Chrome toolbar.

Chrome Web Store installation is not available yet, so keep the extracted folder after installation.

## Included in this release

- EasyList, EasyPrivacy, Russian, social-widget, anti-adblock, and cryptomining protection;
- local tab activity scores with reasons and recommendations;
- reversible Eco Mode for expensive tabs;
- site exceptions and temporary protection pauses;
- cookie-banner and distraction controls;
- element picker, custom rules, and HTTPS filter subscriptions;
- Image Swap, Picture-in-Picture, and optional cookie export;
- separate English and Russian settings;
- local settings backup and restore.

## Privacy

Browser Monitor has no account, advertising SDK, or developer-operated analytics server. Tab measurements and settings stay in the local Chrome profile. Optional cookie, download, and clipboard permissions are requested only when the related tool is used.
